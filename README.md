# test-harness-vCoolStuff

Coding eval harnesses for multiple LLM providers, all using the same set of 10 TypeScript software engineering tasks run inside Claude Code's agentic harness.

Each subdirectory is a self-contained eval for a specific model — see the README inside each for details:

| Directory | Model |
|---|---|
| [test-harness-claude](test-harness-claude/) | Claude |
| [test-harness-glm](test-harness-glm/) | GLM |
| [test-harness-kimi](test-harness-kimi/) | Kimi |
| [test-harness-minimax](test-harness-minimax/) | MiniMax |
| [test-harness-qwencode](test-harness-qwencode/) | Qwen Coder |
